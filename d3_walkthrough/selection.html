With D3 we select nodes using standard W3C Selectors API

Example:

One

Two

Three

Four

HTML 
<p>One</p>
<p>Two</p>
<p>Three</p>
<p>Four</p>
JavaScript 
function applyD3Selectors(){
	d3.selectAll("p").style("color", "white");
	d3.selectAll("p").style("background-color", "black");
}
