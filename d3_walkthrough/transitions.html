From the D3 Documentation: "D3 is not a new graphical representation. Unlike Processing, Raphael, or Protovis, the vocabulary of marks comes directly from web standards: HTML, SVG (Scalable Vector Graphics) and CSS. For example, you can create SVG elements using D3 and style them with external stylesheets. You can use composite filter effects, dashed strokes and clipping. If browser vendors introduce new features tomorrow, you'll be able to use them immediately - no toolkit update required. And, if you decide in the future to use a toolkit other than D3, you can take your knowledge of standards with you!"

The first argument d in the functions below is the DOM object in the DOM query being iterated over. The second argument i is the index of the DOM object; we can use it to give elements unique style.

Example:

100px left margin

130px left margin

160px left margin

190px left margin

d3.selectAll("p").transition()
	    .duration(750)
	    .delay(function(d, i) { return i * 10; })
	    .style("margin-left", function(d,i) { return (100+30*i)+"px"; });
